# Flooding Alert System (FAS)
Image processing algorithm for detecting floods

## Requirements
* NumPy
* OpenCV
        
## Usage
This algorithm consists of several stages:<br />
* Raw input image<br />
* Color segmentation<br />
* Morphological operations<br />
* Shape detection<br />
